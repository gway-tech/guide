이미지 편집

디스크 목록에서 "추가" 또는 "편집"을 클릭하여 디스크 속성 설정 대화 상자를 엽니다. 이름: 디스크의 이름입니다. 범주: 사용자 정의 디스크 분류 유형: 데이터 디스크 및 시스템 이미지를 포함합니다. 클라이언트 드라이브 문자: 클라이언트에 표시되는 디스크의 드라이브 문자 스토리지 리소스: 파티션: 서버 디스크 파티션. 디스크 파일: 가상 디스크 파일, 시스템 이미지는 가상 디스크 파일만 가능합니다. 복사: 새 디스크 파일을 생성할 때 포맷된 샘플 파일에서 복사할 수 있습니다. 용량: 새로운 디스크 파일의 용량을 설정합니다. 새 디스크 파일 생성: 디스크 파일을 다시 생성합니다.
