템플릿 소개

템플릿은 컴퓨터 기능을 관리하는 기술로, 사용자가 컴퓨터 기능 정책을 신속하게 구성하고 악성 소프트웨어, 네트워크 공격, 데이터 유출 및 기타 위협으로부터 컴퓨터를 보호할 수 있도록 도와줍니다. 템플릿에는 다음과 같은 특징이 있습니다.

  • 간단하고 사용하기 쉬움: 템플릿은 그래픽 인터페이스를 제공하며 사용자는 클릭하고 선택하기만 하면 컴퓨터 기능 구성을 완료할 수 있습니다.
  • 효율적이고 유연함: 템플릿은 여러 컴퓨터 그룹을 사용할 수 있도록 지원하며, 사용자는 다양한 컴퓨터 유형과 목적에 따라 다양한 기능 정책을 선택할 수 있습니다. 템플릿은 실행 취소 및 재사용도 지원하므로 사용자는 언제든지 컴퓨터 기능 설정을 수정하고 복원할 수 있습니다.
  • 공유 및 협업: 템플릿은 다른 사용자나 부서와 공유할 수 있으며, 사용자는 네트워크나 기타 방법을 통해 템플릿을 사용해야 하는 다른 사람에게 템플릿을 보낼 수 있습니다. 템플릿은 템플릿 그룹으로 구성할 수도 있으며, 사용자는 여러 관련 템플릿을 함께 모아 관리 및 사용이 편리합니다.

템플릿 기능

템플릿에는 다음과 같은 기능 설정 측면이 포함되어 있습니다.

  • 이미지 구성: 템플릿은 시스템 디스크 및 데이터 디스크, 복원 모드, 유효 기간 및 사용 가능 주 등과 같은 컴퓨터의 이미지 매개변수를 설정할 수 있습니다. 이미지 매개변수는 컴퓨터의 기본 기능과 성능을 결정합니다.
  • 네트워크 구성: 템플릿은 IP 주소, DNS 서버, DHCP 서버 등과 같은 컴퓨터의 네트워크 매개변수를 설정할 수 있습니다. 네트워크 매개변수는 컴퓨터의 네트워크 연결 및 통신을 결정합니다.
  • 고급 구성: 템플릿은 U 디스크 액세스 모드, 주변 장치 액세스 권한, 입력 방법 및 캐시 등과 같은 컴퓨터의 고급 매개변수를 설정할 수 있습니다. 고급 매개변수는 컴퓨터의 보안과 신뢰성을 결정합니다.
  • 디스플레이 구성: 템플릿은 해상도, 색상, 밝기, 다중 모니터 조합 모드 등과 같은 컴퓨터의 디스플레이 매개변수를 설정할 수 있습니다. 디스플레이 매개변수는 컴퓨터의 디스플레이 효과와 시각적 경험을 결정합니다.
  • 워터마크 구성: 템플릿은 워터마크 내용, 위치, 색상 등과 같은 컴퓨터의 워터마크 매개변수를 설정할 수 있습니다. 워터마크 매개변수는 컴퓨터의 소유권과 속성을 결정합니다.
  • 업데이트 구성: 템플릿은 업데이트 모드, 업데이트 시간, P2P 모드 등과 같은 컴퓨터의 업데이트 매개변수를 설정할 수 있습니다. 업데이트 매개변수는 컴퓨터의 보안과 안정성을 결정합니다.
  • 개인 구성: 템플릿은 개인 문서 전송 등 컴퓨터의 개인 매개변수를 설정할 수 있습니다. 개인 매개변수는 컴퓨터의 모양과 기능을 결정합니다.
  • 스위치 시스템 구성: 템플릿은 부팅 시간, 종료 시간, 잠금 화면 시간, 부팅 일괄 처리 등과 같은 컴퓨터의 스위치 시스템 매개변수를 설정할 수 있습니다. 스위치 시스템 매개변수는 컴퓨터의 전원과 리소스를 결정합니다.

템플릿 사용

이 템플릿의 사용 방법은 다음과 같습니다.

  • 템플릿 생성: 사용자는 "새로 만들기" 버튼을 클릭하여 템플릿 생성 인터페이스에 들어가 템플릿 이름과 설명을 입력한 다음 구성해야 하는 기능을 선택하고 안내 메시지에 따라 설정한 후 마지막으로 "확인" 버튼을 클릭하여 템플릿 생성을 완료할 수 있습니다.
  • 템플릿 수정: 사용자는 "편집" 버튼을 클릭하여 템플릿 편집 인터페이스로 들어가 수정이 필요한 기능을 선택하고 프롬프트에 따라 수정한 후 마지막으로 "확인" 버튼을 클릭하여 템플릿 수정을 완료할 수 있습니다.
  • 템플릿 저장: 사용자는 "확인" 버튼을 클릭하여 현재 템플릿을 로컬 또는 네트워크에 저장할 수 있습니다. 사용자는 저장 위치와 이름을 사용자 정의하거나 원본 템플릿을 덮어쓸 수 있습니다.
  • 템플릿 공유: 사용자는 "새로 만들기"를 클릭하여 템플릿 공유를 생성하고 현재 템플릿을 다른 사용자 또는 부서와 공유할 수 있습니다. 사용자는 공유 모드와 상태를 선택하고 공유 지침과 권한을 추가할 수 있습니다.
  • 템플릿 구성: 사용자는 "새로 만들기"를 클릭하여 템플릿 그룹을 생성하고 현재 템플릿을 기존 또는 새 템플릿 그룹에 추가할 수 있습니다. 사용자는 템플릿 그룹 이름과 설명을 사용자 정의할 수 있으며, 템플릿 그룹에서 템플릿을 수정하거나 삭제할 수 있습니다. 템플릿 그룹을 이용하면 사용자는 관련된 여러 템플릿을 쉽게 관리하고 사용할 수 있습니다.
  • 템플릿 적용: 사용자는 "활성화" 버튼을 클릭하여 현재 또는 저장된 템플릿을 하나 이상의 컴퓨터 그룹 및 서버에 적용할 수 있습니다.

이미지 템플릿

이미지 템플릿 개요

이미지 템플릿은 이미지 관리를 위한 기술로, 사용자가 다양한 컴퓨터 구성 요구에 따라 다양한 이미지 파일, 데이터 디스크 및 복원 모드를 선택하는 데 도움이 됩니다. 이미지 파일은 운영 체제, 소프트웨어 및 데이터가 포함된 디스크 파일을 의미합니다. 데이터 디스크는 사용자 데이터를 저장하는 데 사용되는 디스크 파티션 또는 디스크 파일을 의미합니다. 복원 모드는 컴퓨터를 켜고 끌 때 이미지 파일과 데이터 디스크가 처리되는 방식을 나타냅니다. 이미지 템플릿에는 다음과 같은 특징이 있습니다.

  • 유연하고 다양함: 사용자는 다양한 시나리오 내용과 모드에 적응하기 위해 필요한 기능에 따라 다양한 이미지 파일, 데이터 디스크 및 복원 모드를 선택할 수 있습니다. 예를 들어, Windows 또는 Linux 시스템을 선택할 수 있습니다. 국내 운영체제; 다른 교육 소프트웨어를 설치하도록 선택할 수 있습니다. 귀하는 사용자 데이터를 유지하거나 삭제하도록 선택할 수 있습니다. 등.
  • 효율적이고 편리함: 사용자는 네트워크나 기타 방법을 통해 이미지 템플릿을 다른 컴퓨터로 보내 신속하게 구성하고 업데이트할 수 있습니다. 또한 사용자는 이미지 템플릿의 유효 기간과 사용 가능 주를 설정하여 이미지 템플릿을 자동으로 적용하고 취소할 수 있습니다.
  • 공유 및 협업: 사용자는 자신이 생성하거나 사용한 이미지 템플릿을 다른 사용자 또는 부서와 공유하여 리소스 공유 및 지식 상속을 달성할 수 있습니다. 사용자는 쉽게 관리하고 사용할 수 있도록 여러 관련 이미지 템플릿을 이미지 그룹으로 구성할 수도 있습니다.

이미지 템플릿 기능

이미지 템플릿에는 다음과 같은 기능 설정 측면이 포함됩니다.

  • 이미지 파일: 사용자는 기존 또는 자체 제작한 이미지 파일을 컴퓨터 시스템 디스크로 선택할 수 있습니다. 이미지 파일에는 컴퓨터의 기본 기능과 성능을 결정하는 운영체제, 소프트웨어, 데이터 등이 포함되어 있습니다.
  • 데이터 디스크: 사용자 데이터를 저장하는 공간으로 데이터 디스크를 컴퓨터에 할당할지 여부를 사용자가 선택할 수 있습니다. 데이터 디스크의 내용은 이미지 파일의 영향을 받지 않으며, 보관하거나 삭제할 수 있습니다.
  • 복원 모드: 사용자는 컴퓨터를 켜고 끌 때 이미지 파일과 데이터 디스크를 처리하는 방법을 선택할 수 있으며 다음과 같은 여러 옵션이 있습니다.
    • 매번 복원: 컴퓨터는 전원을 켜고 끌 때마다 이미지 파일을 초기 상태로 복원하여 모든 수정 사항과 데이터를 지웁니다.
    • 복원 안 함: 컴퓨터를 켜고 끌 때 컴퓨터는 이미지 파일과 데이터 디스크에 대해 어떤 작업도 수행하지 않고 모든 수정 사항과 데이터를 유지합니다.
    • 매주 복원: 컴퓨터는 일주일에 한 번(사용자 정의 가능) 이미지 파일을 초기 상태로 복원하여 모든 수정 사항을 지웁니다.
    • 매월 복원: 컴퓨터는 한 달에 한 번(사용자 정의 가능) 이미지 파일을 초기 상태로 복원하여 모든 수정 사항을 지웁니다.

네트워크 템플릿

네트워크 템플릿 개요

네트워크 템플릿은 네트워크 관리를 위한 기술로, 사용자가 컴퓨터 구성에 대한 다양한 요구에 따라 다양한 네트워크 매개변수를 선택하는 데 도움을 줄 수 있습니다. 네트워크 매개변수는 컴퓨터의 서브넷 마스크, 게이트웨이, DNS 서버 등을 말하며 컴퓨터의 네트워크 연결 및 통신을 결정합니다. 네트워크 템플릿에는 다음과 같은 특징이 있습니다.

  • 유연하고 다양함: 사용자는 기능 요구에 따라 다양한 네트워크 매개변수를 선택하여 다양한 시나리오 내용과 모드에 적응할 수 있습니다. 예를 들어 고정 또는 동적 IP 주소를 선택할 수 있고, 다른 서브넷 마스크와 게이트웨이를 선택할 수 있으며, 다른 DNS 서버 등을 선택할 수 있습니다.
  • 효율적이고 편리함: 사용자는 네트워크 또는 기타 방법을 통해 네트워크 템플릿을 다른 컴퓨터로 보내 신속하게 구성하고 업데이트할 수 있습니다. 또한 사용자는 네트워크 템플릿의 유효 기간과 사용 가능 주를 설정하여 네트워크 템플릿을 자동으로 적용하고 취소할 수 있습니다.
  • 공유 및 협업: 사용자는 자신이 생성하거나 사용한 네트워크 템플릿을 다른 사용자 또는 부서와 공유하여 리소스 공유 및 지식 상속을 달성할 수 있습니다. 사용자는 쉽게 관리하고 사용할 수 있도록 여러 관련 네트워크 템플릿을 네트워크 그룹으로 구성할 수도 있습니다.

네트워크 템플릿 기능

네트워크 템플릿에는 다음과 같은 기능 설정 측면이 포함됩니다.

개인 구성 템플릿

개인 구성 템플릿 개요

개인 구성 템플릿은 개인 구성 관리를 위한 기술로, 사용자가 컴퓨터 구성에 대한 다양한 요구에 따라 다양한 개인 매개변수를 선택하는 데 도움을 줄 수 있습니다. 개인 매개변수란 컴퓨터 등의 개인 문서 전송을 말하며 컴퓨터의 외관과 기능을 결정합니다. 개인 구성 템플릿에는 다음과 같은 특징이 있습니다.

  • 유연하고 다양함: 사용자는 기능 요구에 따라 다양한 개인 매개변수를 선택하여 다양한 시나리오 내용과 모드에 적응할 수 있습니다. 예를 들어, 개인 문서 등의 다른 전송을 선택할 수 있습니다.
  • 효율적이고 편리함: 사용자는 네트워크나 기타 방법을 통해 개인 구성 템플릿을 다른 컴퓨터로 보내 신속하게 구성하고 업데이트할 수 있습니다. 또한 사용자는 개인 구성 템플릿의 유효 기간과 사용 주를 설정하여 개인 구성 템플릿을 자동으로 적용하고 취소할 수 있습니다.
  • 공유 및 협업: 사용자는 자신이 만들거나 사용한 개인 구성 템플릿을 다른 사용자 또는 부서와 공유하여 리소스 공유 및 지식 상속을 달성할 수 있습니다. 또한 사용자는 쉽게 관리하고 사용할 수 있도록 여러 관련 개인 구성 템플릿을 개인 구성 그룹으로 구성할 수 있습니다.

개인 구성 템플릿에는 다음과 같은 기능 설정 측면이 포함됩니다.

  • 개인 디스크에 사용자 구성 저장: 사용자는 사용자 구성을 개인 디스크에 저장할지 여부를 선택할 수 있으며, 사용자 구성은 일반적으로 데스크톱, 문서, 사진, 비디오, 음악 등을 포함하여 C:\Users 디렉터리에 있는 모든 파일을 참조할 수 있습니다. 전체적으로 전송은 되지만 호환성이 별로 좋지 않아 문서만 전송하는 것을 권장합니다.
  • 슈퍼 사용자에도 저장: 사용자는 사용자 구성을 슈퍼 사용자의 개인 디스크에도 저장할지 여부를 선택할 수 있습니다. 이 옵션을 선택하면 사용자 구성도 슈퍼 사용자의 개인 디스크에 저장됩니다.

  • 개인 디스크 목록으로 파일 전송: 사용자는 개인 디스크 목록으로 전송해야 하는 파일을 선택할 수 있으며, 여러 파일은 여러 줄로 표시됩니다.
