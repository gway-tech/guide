전체 프레젠테이션

웹 인터페이스 로그인 포털은 http://localhost:9000 또는 https://localhost:9443입니다. 기본 비밀번호는 123456입니다.

왼쪽에는 리소스 관리, 서버 관리, 템플릿 관리, 시스템 관리 및 정보가 포함된 기능 탐색이 있습니다.

자원 관리

리소스 관리에는 현재 서버의 하드웨어 상태 모니터링, 클라이언트 시스템의 기록 트래픽, 클라이언트 시스템의 하드웨어 상태 및 이미지 업데이트 상태에 대한 차트 정보가 포함됩니다.

서버 관리

서버 관리에는 이미지 관리, 클라이언트 컴퓨터 목록, 로밍 사용자 목록, 서버 디스크 캐시 목록 및 범주 관리가 포함됩니다.

템플릿 관리

템플릿 관리는 템플릿을 관리하는 것입니다. 템플릿은 컴퓨터에 일괄적으로 적용할 수 있는 기능 모음입니다. 여기에는 이미지 템플릿, 네트워크 템플릿, 고급 템플릿, 디스플레이 템플릿, 워터마크 템플릿, 업데이트 템플릿, 개인 구성 템플릿, 전원 켜기/끄기 템플릿, 템플릿 그룹 및 템플릿 공유가 포함됩니다.

시스템 관리

시스템 관리는 주로 현재 서버의 옵션을 설정하고 비밀번호를 변경하는 데 사용됩니다.

정보

정보에는 현재 서버의 인증 정보가 포함되어 있습니다.
